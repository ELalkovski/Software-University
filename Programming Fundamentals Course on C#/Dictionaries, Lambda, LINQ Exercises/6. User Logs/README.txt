This program reads an input in the format:
IP=(IP.Address) message=(A&sample&message) user=(username)

The program parses the IP and the username from the input and for every user, it displays every IP from which the corresponding user has sent a message and the count of the messages sent with the corresponding IP. 
In the output, the usernames are sorted alphabetically while their IP addresses is displayed in the order of their first appearance. 
The output is in the following format:
username: 
IP => count, IP => count.
